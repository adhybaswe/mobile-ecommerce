# Screenshots Directory

This directory contains screenshots of the mobile e-commerce app.

## Required Screenshots

Please add the following screenshots to this directory:

1. **home.png** - Home screen showing product catalog, search bar, and banner
2. **product-detail.png** - Product detail page with image, description, and add to cart
3. **cart.png** - Shopping cart with items and checkout button
4. **login.png** - Login screen with social login options
5. **checkout-success.png** - Checkout success screen

## Screenshot Guidelines

- **Format:** PNG or JPG
- **Orientation:** Portrait (mobile view)
- **Resolution:** Recommended 1080x2340 or similar mobile resolution
- **Quality:** High quality, clear and readable
- **Content:** Show actual app content, not placeholder data if possible

## How to Take Screenshots

### iOS Simulator
1. Run the app in iOS Simulator
2. Press `Cmd + S` to save screenshot
3. Screenshots are saved to Desktop by default

### Android Emulator
1. Run the app in Android Emulator
2. Click the camera icon in the emulator toolbar
3. Or press `Cmd + S` (Mac) / `Ctrl + S` (Windows)

### Physical Device
1. Use Expo Go to run the app
2. Take screenshots using device screenshot function
3. Transfer to computer
